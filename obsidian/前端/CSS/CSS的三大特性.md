## 特殊性

| 选择器的类型            | 实例                          | 选择器权重 | 等级   |
| ----------------- | --------------------------- | ----- | ---- |
| !important        | div                         | 无穷大   | 特级   |
| 行内样式（style 属性中样式） | style='color:red;'          | 1000  | 第一等级 |
| id 选择器            | \#id                        | 0100  | 第二等级 |
| class、伪类、属性选择器    | .box、:hover、\[type='text'\] | 0010  | 第三等级 |
| 标签选择器、伪元素选择器      | div、::after、::before        | 0001  | 第四等级 |
| 通配符、子选择器、相邻选择器等   | *、> 、+、~                    | 0000  |      |
| 继承的样式             |                             | 0000  |      |
